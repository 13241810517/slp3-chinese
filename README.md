## 自然语言处理综述第三版 中文翻译项目

第二版的中文版在京东有售，但是第二版在05年已经出版，时至今日NLP领域已经发生的巨大变化。在18年年底，作者推出了第三版，英文版地址在：http://web.stanford.edu/~jurafsky/slp3/ed3book.pdf。
